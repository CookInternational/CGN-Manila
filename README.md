## 13. Emergency Troubleshooting

### If media assets do not display

Check file paths, asset names, GitHub Pages case sensitivity, and the published branch.

### If the media page does not load

Check `index.html`, `CNAME`, repository Pages settings, and `media.cgnnews.net` DNS.

### If search engines show stale metadata

Check `sitemap.xml`, `robots.txt`, page meta tags, and the GitHub Pages deployment status.

### If someone expects article publishing here

Route article publishing back to the main CGN News repository and backend workflow. The media repository is not the article source of truth.
